# WebSocket Ping Pong App

A simple real-time WebSocket application to understand
bi-directional communication between client and server.

## Tech Stack
- Node.js
- WebSocket (ws / socket.io)
- HTML / JS

## How it works
- Client sends "ping"
- Server responds with "pong"
- Demonstrates real-time messaging

## Future Improvements
- Convert to chat application
- Add multiple clients
- Add rooms and user names
